Cirrus Framework


Cirrus framework is "a gorgeous css framework for the web", "A fully responsive and comprehensive CSS framework with beautiful controls and simplistic structure. Integrate it with any existing design or use it to kick start your next project". It is said Cirrus doesn't need jQuery, but it is needed if you use anything with Javascript in it.

Alpha 1 of Cirrus was released on January 18th 2017, and has had big updates that come out sporadically.

It is supported on the last 3 version of Firefox, Chrome, Safari, Opera and Electron. As well as Internet Explorer 9, 10, 11 and Edge.

You can find Cirrus at spiderpig86.github.io/Cirrus/, Github, npm, and JSDelivr.

The idea for this came from 1MB Club. Creating a number of webpages that are all less than 1mb in size.
