Tachyons Framework


Tachyons framework is designed as "functional css for humans" and while it is definitely functional the class names are hard to remember so you may have to constantly refer back to the documentation for all the class names. The framework does have a number of principles which you can find on their Github, there you can also find the features list.

You can find Tachyons at tachyons.io, Github, npm, and JSDelivr.

The idea for this came from 1MB Club.
