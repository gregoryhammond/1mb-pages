MUI Framework



MUI is a lightweight CSS framework that follows Google's Material Design guidelines. Of course it has some features, small footprint (they say the css is 6.6 kB gzipped, and the javascript is 5.4 kB gzipped), customizable (Customize breakpoints, font-settings and use Material Design colors), and it doesn't require any external dependencies. There are also options for React, Angular, and HTML email.

MUI is tested (and works on), Internet Explorer 10 and above, the latest stable version of Firefox, Chrome, Safari, and iOS version 6 and above.

You can find MUI at muicss.com, Github, npm, and jsDelivr.

The idea for this came from 1MB Club. Creating a number of webpages that are all less than 1mb in size.
