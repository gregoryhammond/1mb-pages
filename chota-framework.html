Chota Framework


Chota is a CSS framework that is around 3kb, which could be considered micro.

Chota's first release was March 31st 2017, version 0.1.0, with version 0.2.0 coming out the next day. New versions were released every everyday until version 0.3.3, then come out in September of the same year.

The browser support is very limited, for a majority of the browsers it's the last 2 versions, with IE / Edge it's only Edge 16 or higher. I assume this is done so there doesn't have to be many tests done to ensure all browsers support the features added.
The idea for this came from 1MB Club.

My challenge is to make a number of webpages that are 1MB or less in size. I will be creating a number of different pages in different ways.
