Picnic CSS


There is very little information about Picnic CSS other than what is already on their github repo or their website.

The repo doesn't even contain the very first release, it starts at version 3.0 which was released on Febuary 4th 2015. The releases don't even contain the .css file that is the framework, you would either need to download the entire source code or use a CDN that lets you point to an old version.
