Milligram Framework


Milligram framework version 1.0.0 was released on github on December 26th 2015 and has only grown since. It has been kept a small framework on purpose.

It was created by and is still maintained by CJ Patoilo.
The idea for this came from 1MB Club.

My challenge is to make a number of webpages that are 1MB or less in size. I will be creating a number of different pages in different ways.
