Spectre.css Framework


Spectre.css says it's a "lightweight, responsive, and modern CSS framework". It says it's also 10kB when gzipped and it's flexbox-based. The first version was published to npm in April 2016, and has had updates since.

In the Github repo readme, it says that they try to "make most styles compatible with earlier browsers", but it is designed for modern browsers. They have recommend the last 4 versions of Chrome, Microsoft Edge, Safari, and Opera. As well as the extended support release of Firefox, as well as Internet Explorer version 10 or above.

You can find Spectre on Github, cdnjs, npm, JSDelivr.

The idea for this came from 1MB Club. Creating a number of webpages that are all less than 1mb in size.
