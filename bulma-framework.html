Bulma Framework


Bulma is considered a "modern CSS framework based on Flexbox". It only requires one CSS file, it is compatible with only the most recent versions of Chrome, Edge, Firefox, Opera, and Safari. IE 10 and above are only partially supported.

The very first release on Github was on January 24th 2016 and at the time of writing is on version 0.9.1 which was released September 28th 2020.

It is not shown how much money is made by the creator, and it is none of our business. But it does make a good amount since there is a good amount of names on the backers page.
