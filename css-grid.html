CSS Grid

CSS grid is a 2D grid-based layout system. It doesn't require any additional frameworks or libraries. At the time of writing CSS Grid is supported by 96.09% of all browsers according to Can I Use.



Instead of re-writing what already has been written about the history of CSS Grid, you can read the detailed article on the history of CSS Grid at A List Apart.
